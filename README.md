# Task Tracker API

A RESTful API to track and manage your tasks, built with Spring Boot. This project helps practice backend development skills including REST API design, JPA/Hibernate, and database operations.

## Features

- Add, update, and delete tasks
- Mark tasks as "in progress" or "done"
- List all tasks
- Filter tasks by status (done, not done, in progress)
- Persistent storage using MySQL database

## Tech Stack

- Java 17+
- Spring Boot 3.x
- Spring Data JPA
- MySQL
- Maven

## Requirements

- Java 17 or higher
- MySQL 8.0+
- Maven 3.6+
